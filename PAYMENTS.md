# Настройка платежных систем

Бот поддерживает три способа оплаты:

1. **Telegram Stars** (встроенный)
2. **YooKassa** (российский платежный шлюз)
3. **CryptoPay** (криптовалюты через @CryptoBot)

## Тарифы

В боте настроены следующие тарифы:
- **Пробный период**: 3 дня (бесплатно)
- **1 месяц**: 300 рублей
- **3 месяца**: 800 рублей
- **6 месяцев**: 1400 рублей

Цены можно изменить в файле `.env`.

## 1. Telegram Stars

### Настройка

1. Включите Telegram Stars в `.env`:
```
USE_TELEGRAM_STARS=true
```

2. Цены для Telegram Stars указываются в XTR (Telegram Stars).

### Преимущества
- Встроенная система Telegram
- Не требует дополнительной регистрации
- Мгновенное зачисление

### Недостатки
- Доступно только для пользователей с Telegram Premium
- Ограниченная география

## 2. YooKassa

### Регистрация

1. Зарегистрируйтесь на [yookassa.ru](https://yookassa.ru)
2. Создайте магазин
3. Получите `shopId` и `secretKey` в личном кабинете

### Настройка

1. Добавьте данные в `.env`:
```
YOOKASSA_SHOP_ID=your_shop_id
YOOKASSA_SECRET_KEY=your_secret_key
```

2. Настройте webhook URL в личном кабинете YooKassa:
```
https://YOUR_DOMAIN/webhook/yookassa
```

### Поддерживаемые методы оплаты
- Банковские карты (Visa, MasterCard, МИР)
- СБП (Система быстрых платежей)
- ЮMoney
- Qiwi Wallet
- Альфа-Клик
- Сбербанк Онлайн

### Комиссия
От 2.8% + 10₽ за операцию (зависит от тарифа)

## 3. CryptoPay

### Настройка

1. Создайте приложение в [@CryptoBot](https://t.me/CryptoBot)
2. Получите API токен
3. Добавьте токен в `.env`:
```
CRYPTOPAY_TOKEN=your_cryptopay_token
```

### Поддерживаемые криптовалюты
- USDT (Tether)
- TON (The Open Network)
- BTC (Bitcoin)
- ETH (Ethereum)
- BNB (Binance Coin)
- USDC (USD Coin)

### Преимущества
- Низкие комиссии
- Быстрые транзакции
- Анонимность
- Глобальная доступность

### Комиссия
1% за операцию

## Процесс оплаты

1. Пользователь выбирает тариф подписки
2. Бот показывает доступные способы оплаты
3. Пользователь выбирает способ оплаты
4. Создается счет на оплату
5. После успешной оплаты бот автоматически создает подписку в 3x-ui
6. Пользователь получает данные для подключения

## Безопасность

- Все платежи обрабатываются через официальные API платежных систем
- Бот не хранит данные банковских карт
- Webhook запросы проверяются на подлинность
- Все транзакции логируются в базе данных

## Тестирование

Перед запуском в продакшен обязательно протестируйте все способы оплаты:

1. Используйте тестовые ключи API (если доступны)
2. Проверьте создание счетов
3. Проверьте обработку успешных платежей
4. Проверьте обработку отмененных платежей
5. Проверьте создание подписок после оплаты

## Мониторинг

Отслеживайте следующие метрики:
- Конверсию в оплату по каждому методу
- Средний чек
- Отказы от оплаты
- Технические ошибки при оплате

Все платежи сохраняются в таблице `payments` базы данных для анализа.
