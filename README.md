# 3x-ui VPN Telegram Bot

Телеграм бот для автоматизации продажи и управления подписками VPN через панель 3x-ui.

## Основные функции

- 🚀 **Управление подписками**: Автоматическое создание и удаление клиентов в 3x-ui.
- 💳 **Мультиплатежность**: Поддержка Telegram Stars, ЮKassa и CryptoPay.
- 🎁 **Пробный период**: Возможность создания бесплатной подписки на 3 дня.
- 📊 **Тарифная сетка**: 1 месяц (300₽), 3 месяца (800₽), 6 месяцев (1400₽).
- 📱 **Личный кабинет**: Информация об активной подписке и ссылка на подключение.
- 🛡️ **Админ-панель**: Статистика и ручное управление пользователями.

## Требования

- Python 3.8+
- Установленная панель [3x-ui](https://github.com/MHSanaei/3x-ui)
- Токен бота от [@BotFather](https://t.me/BotFather)

## Быстрая установка

Мы подготовили скрипт для автоматической установки:

```bash
wget https://raw.githubusercontent.com/antohal2/mybot/main/install.sh
chmod +x install.sh
./install.sh
```

Скрипт создаст виртуальное окружение, установит зависимости и настроит системную службу (systemd).

## Ручная настройка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/antohal2/mybot.git
cd mybot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте конфигурацию:
   Переименуйте `.env.example` в `.env` и заполните необходимые поля (токен бота, данные панели, ключи платежных систем).

4. Запустите бота:
```bash
python bot.py
```

## Структура проекта

- `bot.py` — Главный файл бота и обработчики команд.
- `config.py` — Загрузка и валидация конфигурации.
- `database.py` — Работа с базой данных (SQLite).
- `xui_client.py` — Взаимодействие с API 3x-ui.
- `payments.py` — Интеграция с платежными шлюзами.
- `install.sh` — Скрипт автоматизации развертывания.

## Администрирование

Админы (указанные в `ADMIN_IDS` в `.env`) имеют доступ к:
- Кнопке `📊 Админ-панель` с общей статистикой.
- Команде `/add_sub [user_id] [days]` для ручной выдачи подписки.
